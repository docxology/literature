### Integrating Cognitive Map Learning and Active Inference for Planning in Ambiguous EnvironmentsIntegrating cognitive map learning and active inference offers a powerful approach to planning in complex, ambiguous environments. The authors state: “Living organisms need to acquire both cognitive maps for learning the structure of the world and planning mechanisms able to deal with the challenges of navigating ambiguous environments.” This work investigates the potential of active inference as a planning mechanism for cognitive maps, proposing the integration of a statistical model of cognitive map formation within an active inference agent that supports planning under uncertainty. Specifically, the research examines the clone-structured cognitive graph (CSCG) model of cognitive map formation and compares a naive clone graph agent with an active inference-driven clone graph agent in three spatial navigation scenarios. The findings demonstrate that while both agents are effective in simple scenarios, the active inference agent is more effective when planning in challenging scenarios, in which sensory observations provide ambiguous information about location.### MethodologyThe authors describe the mechanisms driving standard clone-structured cognitive graphs for structure learning.
