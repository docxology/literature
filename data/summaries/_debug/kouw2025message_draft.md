The agent utilizes an autoregressive model to predict continuous-valued observations and infer bounded continuous-valued actions. The core contribution is a validated design demonstrated on a robot navigation task, showcasing exploration and exploitation capabilities compared to a classical optimal controller. The agent’s key innovation is its ability to predict the system’s dynamics more accurately, leading to a later but more informed arrival at the goal.### MethodologyThe proposed agent employs a factor graph to model the robot’s dynamics. The model is autoregressive, meaning the system output at time *k* is predicted from the system input *u*, M previous system inputs *ū*, and M previous system outputs *ȳ*. The likelihood function is based on a Gaussian distribution with a regression coefficient matrix *A* and a precision matrix *W*. The agent’s approach is based on the principle of minimizing the difference between the predicted and observed outputs.### ResultsThe experimental results demonstrate that the proposed agent outperforms a classical optimal controller in a robot navigation task.
